電子郵件：laydevin@gmail.com

一、資訊收集

1. 自動收集的資訊

  • 應用使用情況：使用 Firebase Analytics 收集匿名化統計資料，包括應用啟動次數、功能使用情況、效能資料。
  • 裝置資訊：匿名化的裝置型號、作業系統版本等技術資訊。

2. 您主動提供的資訊

  • 閱讀偏好設定：字體大小、行距、閱讀主題等。
  • 註記資料：您在閱讀時建立的螢光筆標記和筆記。
  • 閱讀進度：您在各書籍中的閱讀位置。

3. 本地儲存的資料

  • 下載書籍：從 CBETA 佛典資料庫下載的 EPUB 檔案。
  • 書籍列表與註記資料：儲存在設備本地（JSON 格式或 Room Database）。

4. 不收集的資料

個人身份資訊（姓名、電子郵件、電話號碼等）、位置資訊、聯絡人資訊及其他應用使用情況。

所有收集的資料均為匿名化統計或存於設備本地，無法識別個人身份。

二、資訊使用方式

  • 提供核心功能：顯示書籍內容、管理註記、記錄閱讀進度。
  • 改善應用體驗：分析匿名化使用資料以優化功能。
  • 技術支援：診斷與解決技術問題。

個人閱讀資料（註記、閱讀進度等）僅存於設備本地，不會上傳伺服器。

三、資料儲存與安全

  • 本地儲存：使用 Android SharedPreferences 或 Room Database 儲存書籍與註記資料。
  • 資料安全：採用 Android 系統安全儲存機制，支援系統備份功能。
  • 刪除方式：刪除應用或書籍時，相關資料將一併刪除。

四、第三方服務

  • Firebase Analytics (Google LLC)
    用途：收集匿名化使用統計資料。
    隱私政策：https://policies.google.com/privacy
  • CBETA 佛典資料庫 (中華電子佛典協會)
    用途：提供 EPUB 書籍下載。
    隱私政策：https://www.cbeta.org/
  • Google Text-to-Speech (TTS, Google LLC)
    用途：文字轉語音朗讀功能，主要在設備本地處理。
    隱私政策：https://policies.google.com/privacy

五、應用權限

  • 網路存取 (INTERNET)：下載書籍及同步 OPDS 資料庫。
  • 儲存空間 (READ/WRITE_EXTERNAL_STORAGE)：儲存及讀取下載的 EPUB 書籍檔案（Android 10+ 使用 MediaStore API 可能不需權限）。
  • TTS 查詢 (Android 11+)：查詢可用文字轉語音引擎，提供朗讀功能。

權限僅用於提供核心功能，無額外用途。

六、資料分享與揭露

  • 不會出售、交易或分享您的閱讀內容、註記或個人設定。
  • 僅在法律要求或技術服務提供者需求下，提供匿名化統計資料。

七、兒童隱私

本 App 不針對 13 歲以下兒童設計，亦不會故意收集其個人資訊。如發現孩子提供資訊，請聯絡我們，我們會立即刪除相關資料。

八、使用者權利

  • 資料存取與修改：可在 App 內查看、修改閱讀設定與註記。
  • 資料刪除：刪除已下載書籍或卸載 App 即可刪除所有資料。
  • 資料備份：資料儲存在設備本地，可使用 Android 系統備份功能備份。

九、隱私權政策變更

本政策可能不定期更新，重大變更將在 App 內或透過其他方式通知。建議定期查看政策以了解最新資訊。
